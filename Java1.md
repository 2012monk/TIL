## 접근제어자 Acess Modifier

[참고자료](https://wikidocs.net/232)

---

`public,protected,default,privat` `순서대로 많은 접근을 허용`

1. private
   * 해당 클래스에서만 접근 가능

2. default
   * 접근제어자를 별도로 설정하지 않았을때 해당 패키지 내에서 접근 가능

3. protected
   * 동일 패키지내의 클래스, 해당 클래스를 상속받은 외부 패키지의 클래스에서 접근가능

4. public
   * 어떤 클래스에서도 접근가능
